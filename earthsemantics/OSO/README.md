# OSO namespace at w3id.org

This subpath provides persistent identifiers for the OSO ontology and its related metadata descriptions.

## Maintainer
Steven / @spiel-ifremer  
https://github.com/spiel-ifremer

## Scope
This namespace covers the following persistent identifiers:

- `/earthsemantics/OSO`
- `/earthsemantics/OSO/void`
- `/earthsemantics/OSO/dcat`

## Target project
OSO ontology repository: https://github.com/emso-eric/oso-ontology

## Notes
This subpath is maintained independently under `/earthsemantics/OSO` and does not modify other `earthsemantics` redirects.
